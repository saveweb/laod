Chrome 浏览器修改默认搜索引擎跟网页右键搜索 启动特定页

2016-09-01 14:31

一直以来有不少朋友说我用hosts并且设置了谷歌为默认搜索引擎，但是为什么还是无法访问、连接不上?

无法访问此网站
网址为 http://www.google.com.hk/url?sa=p&hl=zh-CN&pref=hkredirect&pval=yes&q…

的网页可能暂时无法连接，或者它已永久性地移动到了新网址。
ERR_CONNECTION_ABORTED

其实老D早就说过，谷歌、推特等这些网站必须用https形式访问！

如：

谷歌学术：https://scholar.google.com/

谷歌搜索：https://www.google.com/ncr

谷歌香港：https://www.google.com.hk/ncr

网址后面的ncr的意思是：(No Country Redirect)禁止国别跳转，重定向的意思。（Google会进行各种判断从而进行转跳）

但是Chrome 浏览器默认搜索是：普通的http，http://www.google.com 或者http://google.com

它并没有采用https 那么我们就将它修改成https，并且不让它国别跳转。

还有修改启动时默认打开的一个特定网页：

首页打开Chrome浏览器，点击浏览器右上角菜单“自定义及控制Google Chrome ”，然后点击“设置”

如下图：

当然你也可以直接可以在地址栏输入“chrome://settings/”按回车键，也是可以快速转到Chrome浏览器设置页面的。

设置页面里，你可以设置：

Chrome启动时，打开特定网页或一组网页。

你可以设置成喜欢的网页，比如：谷歌搜索或者百度搜索。

如果设置谷歌搜索，那么请填：https://www.google.com/ncr

你也可以设置浏览器显示主页按钮：

接下来修改Chrome浏览器设置默认搜索引擎：

转到“搜索”部分。

然后点击“管理搜索引擎”

分别有三栏：

1.名称 2.关键字 3.搜索引擎网址

需要注意的是：网址（用“%s”代替搜索字词）下面老D给出的三组网址都是带有%s的。

你可以下面一组网址填入：
中文：https://www.google.com/search?hl=zh-CN&q=%s
英文：https://www.google.com/search?hl=en&q=%s
强制加密：https://encrypted.google.com/search?{google:RLZ}{google:acceptedSuggestion}{google:originalQueryForSuggestion}sourceid=chrome&ie={inputEncoding}&q=%s
