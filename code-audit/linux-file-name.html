Linux 大写文件名转小写

2015-10-18 22:03 • 代码审计 • 阅读 3163

很简单的脚本实现

[cc lang=”php”]
for file in * ;
do mv $file `echo $file | tr ‘A-Z’ ‘a-z’`;
done
[/cc]
