Linode被墙SSH无法连接通过Lish连接方法

2018-07-09 13:12

Linode VPS主机是目前市面上较为稳定、性价比以及速度快的VPS商之一,尤其是在提升配置之后使用的用户非常多,口碑一向良好。但是我们使用过程中往往遇到端口被墙的情况，我们可以通过修改端口可以解决这个问题。

使用Mac OS终端应用或PuTTY（Windows系统下）等SSH工具，选择以下数据中心网关地址：

lish-atlanta.linode.com
lish-dallas.linode.com
lish-frankfurt.linode.com
lish-fremont.linode.com
lish-london.linode.com
lish-newark.linode.com
lish-singapore.linode.com
lish-tokyo.linode.com

打开一个终端窗口，输入以下命令，用你Linode管理用户名替换username和您的首选Lish网关地址替换location，Lish端口为22、443和2200。

ssh [email protected]

例如，登录Newark网关用户的样子：

ssh [email protected]

 

输入您用于登录到Linode管理密码，你现在Lish看到你Linodes列表，如下图所示：

Linodes located in this data center:
linode241706 Newark, NJ
linode276072 Newark, NJ
Linodes located in other data centers:
linode287497 Dallas, TX

在Lish命令提示符选择列表的Linode的类型名称；例如：键入linode241706即可将你连接到这台Linode的屏幕控制台会话。
