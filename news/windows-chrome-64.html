Windows 版 Chrome 会在合适的机器上自动切换到 64 位

2017-05-03 22:27

一般来说在条件允许的情况下，用户大多都会尽量去选择 64 位的软件，以此来达到更出色的性能表现。但问题是，肯定还有一部分使用者不清楚这点，而在最新的 Windows 版 Chrome 58 里，Google 就想好要「越俎代庖」了。是说，这个全新的版本，在检测到电脑是运行 64 位系统并拥有不少于 4GB 的内存后，就会自动从 32 位切换到 64 位模式了。

除此之外，macOS 版 Chrome 59 beta 的通知现在也发生了一些变化。首先一点是，通知现在会直接出现在 macOS 的通知中心内，同时其外观也会有改变，各类操作选项呈现的方式也会有所不同，然后图片也变成不会显示了。另外，浏览器会原生支持 APNG 动图，满足更多种类图像的显示需要。
