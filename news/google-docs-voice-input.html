Google Docs 推出语音输入，彻底告别键盘？

2016-02-25 17:06

之前在九月的时候，Google 在 Docs 中推出了用语音输入文字的功能, 让你不用打字就可以轻松写出长篇文章来。用说的是不是比用打字要更有效率还有待商榷，但对于打字不快的人来说，不啻是个福音啊！Google 今天又更进一步地加强了语音输入，推出了让你可以直接在语音里面下编辑指令的新功能，包括换行、开始新段落、加条列点、选取文字、加上文字效果，甚至直接输入颜文字。不过目前这个功能对我们来说还有点遥远，因为它暂时只支持英文，但以 Google 语音辨识之强大，应该只是早晚的问题吧！

近日 Google Docs 在其官方博客上宣布在应用中集成了新款语音听写工具，实际上 Google Docs 早在去年就推出了语音输入功能，允许用户利用语音输入通过 Chrome 浏览器进行文档打字，但当时弊端比较明显，输入精准度不高，而且当出现错误时无法通过语音进行修改，仍需利用键盘输入来修正错误。

还没有试过语音输入的朋友，可以到 Google Docs 的工具里找到它。蛮好玩的呢！
