谷歌：GCP已经准备好进击市场

2016-03-25 16:50 • 业界资讯 • 阅读 2523

谷歌云平台负责人Diane Greene于本周三向CIO们高调宣称，谷歌云平台已经做好全面冲击市场的准备。

近来，在斩获几基关键性胜利之后，Google Cloud Platform对自家产品进一步扩充，并同AWS、微软Azure大打价格战。然而，该公司却一直没能吸引到大型企业客户的关注。

本周三，在旧金山召开的Google Cloud Platform Next 2016大会上，谷歌公司宣称迪士尼与可口可乐两位巨擘已经正式成为其客户。

Greene同时表示，客户选择Google Cloud Platform主要出于GCP更出色的性价比、更低的风险水平、享受创新成果这三大理由。

而就在几个月之前，谷歌公司刚刚与Spotify、苹果建立服务合作，这两家企业将在内部使用Google Cloud Platform。值得一提的是，可口可乐公司也是AWS的客户，而AWS的榜样级客户Netflix公司也在使用Google Cloud Platform。

除了客户数量增加之外，谷歌方面还采取其它多项举措以改进自身Google Cloud Platform。谷歌公司CEO Sundar Pichai最近宣称，Google Cloud Platform已经“准备好应对规模化需求。”

此外，谷歌还公布了谷歌Stackdriver的beta测试版，其将提供统一化监控、日志记录与诊断功能，不仅能够极大改进Google Cloud Platform的整体运行表现，同时也可用于管理AWS账户。

谷歌公司还发布了新的Google Cloud Platform企业级功能，具体包括审计日志记录、身份与访问管理（简称IAM）角色并允许客户为Compute Engine提供加密密钥。另外，谷歌的Cloud Storage也开始进入beta测试。

在Next大会的首日环节当中，数据与机器智能也占据了一席之地，谷歌方面推出了新的Cloud ML（即机器学习）平台以及语音API。

谷歌的BigQuery亦迎来数项更新，该公司同时公布了多款新型云网络工具，包括虚拟专有云工具Subnetworks、Cloud Router以及Cloud CDN。

这套平台最近还在地理层面实现了扩张，其最新数据中心坐落于波特兰与东京，并计划到2017年年末将服务区增加至10个。

VMware公司前任CEO兼联合创始人Greene于2015年11月被谷歌任命为企业云业务负责人，与此同时谷歌还收购了她建立的初创企业Bebop。Greene在应对企业客户方面的丰富经验将帮助谷歌在竞争激烈的云业务市场上取得优势。
