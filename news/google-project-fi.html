Google虚拟运营商网络 Project Fi 覆盖全球！

2016-08-08 18:37

Google虚拟运营商网络 Project Fi 覆盖全球，然而资费依旧让人遐思。

Google 的虚拟运营商网络 Project Fi 通过其 Blog 正式宣布了这个消息。

根据 Google 的消费者调查显示只有 20% 的美国人在出国后使用蜂窝数据，更多的则是选择 Wi-Fi 热点或当地运营商。

而造成这种现象的原因 Project Fi 认为是在出国旅行时网络数据不够快，只能够使用 2G 网络，对这种网络 TechCrunch 表示无法想象如何正常使用 Pokemon GO。

但是现在 Project Fi 网络覆盖十多个国家，135 个以上的地区，这些将包含 97% 美国人出国旅行的地点，比如伦敦、巴黎、柏林、东京和蒙特利尔等城市。

并且因为与新的运营商 3 达成合作，Project Fi 将为海外用户提供比此前蜂窝数据高 10-20 倍的网络速度。

Project Fi 产品经理 Tyler Kugler 表示，用户无需对此额外付费，并且不会有国际漫游费用。

目前 Project Fi 的费用为 20 美元，无限量的国内语音通话和短信，而国际部分虽然没有漫游费，但语音通话时间按分钟计算，不同国家计费不同，以中国为例，每分钟大约 0.2 美元。

Project Fi 同时提供 Wi-Fi 语音通话方式，这种方式价格比较低，同样以中国为例，每分钟 0.01 美元。

网络流量计算方式 10 美元/GB，按量计费，结账时若不到 1GB 则按比例收费。

粗略算下来每月 200 分钟通话，1GB 流量，大概需要 50 美金费用。
