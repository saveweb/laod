项目托管服务Google Code 正式关闭

2016-01-27 00:45

Google的开源项目托管服务Google Code今天正式关闭。Google是在去年3月宣布Google Code将于2016年1月25日（美国时间）关闭，从8月24日（北京时间25日）起进入到只读模式，开发者托管在Google Code上的项目仍然可以通过Google Code Archive下载。

Google Code输给了更受欢迎的GitHub，搜索巨人也建议开发者将项目迁移到GitHub。GitHub正逐渐变成软件行业的开发基础设施。
