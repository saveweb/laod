微软推出 Linux 版 SQL Server 数据库

2016-03-18 20:33

SQL Server 是微软的旗舰关系数据库产品，该数据库一直只提供 Windows 下的版本。今天微软宣布推出可运行在 Linux 系统下的 SQL Server 数据库，该版本目前还是早期未公开的预览版本，计划在 2017年中正式推出。

在 Linux 上运行 SQL Server 数据库将为客户提供更灵活的产品解决方案，微软将很快会公开该版本。过几天微软将在纽约发布 SQL Server 2016 新版数据库，提供更好的内存中数据库、R语言支持以及数据仓库支持，同时更好的支持移动应用开发。

微软称 SQL Server 2016 已经为 Azure 云上的用户提供超过 140 万数据库实例，尽管当前还是预览版，今年晚些时候会变成正式版。

微软推出 Linux 版本的数据库系统对我们来说是难以想象的，自从新 CEO 上任之后，微软从其服务到产品变得越来越开放，而且在开源领域方面也投入越来越多的资源。
