快上 50 倍，Google 给开发者带来了一项新福利

2015-11-24 19:37

对于 Android 开发者或者编程爱好者来说，Android Studio 是一个评价颇高的的开发环境。经过多个版本的迭代后，据 Venturebeat 的消息，现在 Android Studio 即将迎来 2.0 版本。


目前，Google 已经发布了 Android Studio 2.0 预览版。其主要进步在于：

  • Instant Run：只要点击运行，使用者可以立即查看新代码的运行效果。该版的 Instant Run 速度是以往的 50 倍。而以前点击 Instant Run 之后所花费的时间，足够开发者去上个厕所。
  • GPU Profiler 工具：允许使用者记录完整 OpenGL 会话，并逐帧检查所有变动。使用者也能查看每一帧代码，并获得 GL State 和 Commands 信息。
  • 更快的 Android 模拟器
  • 支持使用者将 APK 文件拖拽到模拟器，并能通过拉动窗口调整尺寸
  • 界面获得更新，增加一条悬浮工具条

据称，现在使用 Android Studio 2.0 版为 Android M 开发应用的速度比之前快上了 5 倍。

如果有兴趣体验这个预览版，各位可移步至 Android 开发者网站进行了解。
