或许你每天都在用 Google 的这项服务，却不知道它进行了重大更新

2016-06-18 21:06

除了 Google，似乎没有第二家公司能够如此热衷于将自己付出诸多努力开发改进的产品开放给所有人使用，而且还是完全免费的。

最近，Google 推出了新版的 Web 字体网站 Google Fonts。新版网站共有 804 种字体；这些字体是由 Google 与来自全世界的字体设计师们合力设计的。由于是开源项目，所有这些字体都是免费的，不仅可以直接下载到本地使用，还可以通过相关代码直接嵌入在网页上。

为了更好地展示这些字体，Google 对网站界面了进行了调整。整个网站遵循了 Google 全新的 Material Design 设计规范，网站 logo 也被重新设计。除了添加字体搜索功能，所有字体还依据艺术风格、流行程度、使用语言进行了分类。

804 种字体的每一种都拥有一个独立的网页。网页中详细地展示了字体的相关信息以及设计师的名字和照片。不仅如此，为了更加直观地呈现每一种字体在不同情况的效果，网站还设置了一些操作选项：通过对字体的大小、粗细进行自定义调整，该字体下的示例文本会实时呈现出相应的状态。这对于需要选择字体的网页设计师而言，无比便利。

对此，科技媒体网站 The Verge 的评价是：

新版的 Google Fonts 数据库简直是设计极客的天堂。

实际上早在 2010 年的 I/O 开发者大会上，Google 就首次推出了 Google 字体的 API 接口和用户指引。那一次 Google 只推出了 18 种经过授权的 Web 字体，但都是免费的。要知道，当时已经有 Monotype Web Fonts、FontShop 等诸多字体商，而它们的所提供的 Web 字体服务都是要收费的。

在此后的几年里，Google 继续对 Web 字体数据库进行更新和完善；当然，整个过程中这项服务依然是免费的。再加上 Chrome 浏览器和 Android 系统的影响力，越来越多的网站和网页都开始使用 Google 的 Web 字体服务。到今天，它已经成为整个互联网的基础性服务项目。

我们每天所浏览的网页，就有很大一部分使用了 Google Fonts 的相关服务。

基于 Web 字体的特征，新版 Google Fonts 的愿景是“通过字体让整个 Web 变得更加美观、快速和开放” 。对此，Google 解释称：

我们坚信，唯有伟大的设计和科技，才能够让网站和产品获得个性和体验。我们的目标是通过提供自然、开源的 Web 字体，让整个过程变得更加简单。我们希望这个世界的任何一个用户都能够将我们的 Web 字体天衣无缝地融入到他的设计项目中去。

你现在阅读的这篇文章所采用的字体，就来自于 Google Fonts。
