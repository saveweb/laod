微软必应Bing搜索引擎服务 遭到大规模无法访问！

2019-01-24 13:42

从昨天开始，很多网友就发现 微软旗下的必应（Bing）搜索引擎无法无法打开了，还有的童鞋怀疑是自己的电脑出了问题…

而根据我们做了一些测试：

[mark_b]

bing 国内 IP 被 block （不是被墙） + 域名国内运营商 DNS 停止解析。

验证一：

tracert -d 202.89.233.100

tracert -d 202.89.233.101

这两个国内 IP 被运营商黑名单。

验证二：

nslookup cn.bing.com 你的运营商 DNS

直接返回解析到本机地址 127.0.0.1

解决方法：

路由器设置静态 DNS

cn.bing.com 解析到微软美国 IP 13.107.21.200 即可。

[/mark_b]

一些解决办法：

一、用必应（Bing）临时域名：

www2.bing.com

或者

www4.bing.com

二、修改hosts文件：

用户只需要暂时修改下host然后坐等微软服务器恢复后再删除即可，这样就不需要再更换成其他搜索了。

13.107.21.200 cn.bing.com

修改后如果浏览器依然无法正常打开请直接重启系统，重启后再尝试打开必应搜索看看能否恢复正常访问。

三、修改DNS服务器

把修改成谷歌的：8.8.8.8

或者用Cloudflare的：1.1.1.1

四、全局梯子

既然都有梯子了，我还是用谷歌吧…
