Firefox 57+ 值得推荐的 WebExtension 拓展

2018-03-02 21:08 • 工具软件 • 阅读 16336

Firefox 从 57 版本开始将全面启用 WebExtensions 拓展，传统 XUL 拓展将不被支持，这个决定有利有弊，弊在于拓展功能性降低，利在于 WebExtensions 拓展支持多进程，浏览速度将更加快速流畅，在跨浏览器拓展开发将更容易被移植，WebExtensions 拓展将是未来发展的方向，后续Firefox官方将会提供更多的API，Firefox 拓展也将比其他浏览器的拓展更加强大，更加Firefox。

 

PS: 下面所列出的大部分 WebEtension 拓展均支持所有版本的Firefox，但部分拓展只支持Firefox57+。

拓展推荐

WebExtensions 常用拓展替代
视频下载Video DownloadHelper 替代 NetVideoHunter，支持的绝大多数网站。
代理工具Proxy SwitchyOmega 替代 AutoProxy，相对比较简单。
用户脚本管理Tampermonkey 替代 Greasemonkey，Greasemonkey 4.0 新版并不那么好用。
用户样式管理xStyle 替代 Stylish，Firefox吧出品，Stylish 3.0 新版并不那么好用。
拨号页工具Speed dial 2 替代 Super Start，功能很完善，提供了大部分网站的略缩图。
手形拖拽ScrollAnywhere 替代 Grab and Drag，更加轻量化。
视频广告过滤YiClear 替代 ACVAA，看上去是有团队在维护，失效点击更新即可。
翻译工具轻灵划译，虽然并不那么完美，但还是一款非常不错的翻译拓展。
稍候阅读In My Pocket 替代 ReadItLater，没想到还真有同类WE拓展，速度也很快。
标签设置Tab Mix WebExtension，老牌标签拓展，还在实验开发阶段，功能不多。
下载软件调用ThunderCross 替代 xThunder，安装后需要再次安装一个exe文件。
广告过滤uBlock Origin，完美兼容新版和旧版Firefox。
密码管理LastPass Password Manager，完美兼容新版和旧版Firefox。
视频弹窗同时安装 Maximize Video 和 Popup window，可一键弹出小窗播放，替代火狐一键应用。
图片批量下载Download Star，遗憾不支持视频下载，替代 DownThemAll。
  
功能增强
Youtube Video DownloaderY2B下载工具，支持多种分辨率下载。
AdBlocker for YouTube去除Y2B视频上的广告。
Email Notifier邮件提醒功能，支持所有邮箱。
Page to Panel以手机UA和小窗口模式打开指定网页。
https everywhere只要网站提供 SSL 服务，就自动转到 HTTPS 连接。
OneTab标签页太多了？将您的标签页转换成一个列表
WikiWand让排版老旧的Wikipedia有一个全新的界面。
Second Search引用标记了“关键字”的书签进行搜索。
Url to QrCode页面QRCode生成工具，优势在于支持所有页面。
User-Agent Switcher用户代理切换工具。代理选项非常完善。
FoxyTab标签增强工具。提供各种标签增强功能。
Multiple Tab Handler标签增强工具。批量管理标签页。
Tab Center Redux侧边栏工具。将当前打开的标签在移到侧边栏中显示，支持略缩图。
Tree Style Tab侧边栏工具。将当前打开的标签全部在侧边栏中显示，同能显示标签之间的从属继承关系
Tile Tabs WE标签页分割同屏显示，非常适合于同时进行多项任务的情况。
Dark Mode黑夜模式。将网页变成黑夜阅读模式。
Selection Context Search右键搜索工具。同类拓展里功能最多的，因API限制，无法使用搜索栏中的引擎。
Snap Links Plus批量打开链接工具。一次性选择多个链接同时打开。
Update Scanner监测跟踪网站更新工具。用于跟踪没有提供RSS的网站更新。
Reverse Image Search图片以图搜图工具。
Bookmarks Organizer书签整理工具。可以一键查找失效书签、重复书签、检测书签是否重定向并自动修复。
BriefRSS本地阅读器。老牌本地RSS阅读器的WE拓展版。
Save In…快速保存工具。包括图片，视频，音频，还有已选文字到txt文档。
Textarea Cache自动保存输入框内已输入的文字。
Video Speed ControllerHTML5视频全局控制播放工具。ZX为后退前进10秒，SD为减慢加快播放速度，R为恢复默认。
Link Text and Location Copier链接复制粘贴工具，支持多种格式，支持自定义。
Enable Right Click and Copy右键解锁工具。包括选择，复制，右键菜单等
云盘万能钥匙自动输入百度云分享文件的密码，还能提前显示分享的链接是否失效。
CopyfishOCR文字识别工具。支持图片视频PDF，通过截图的方式进行认别。
Enable Right Click and Copy解除网页右键限制。在https加密网页上也能使用。
Search Multi Tabs同时搜索多个标签的关键词 。
Block Site让某一些网站无法访问，适合工作的时候使用。
斗鱼HTML5播放器做为第一大直播平台，没有HTML5播放器，实在说不过去。
Forget that page一键清理当前网页痕迹,，包括 cookies、历史记录、本地存储、会话存储内容。
  
  
  
开发者拓展
开发者工具箱开发必备，Shift+Ctrl+Alt+I 弹出开发者工具箱，替代DOM Inspector和Firebug。
Firefox Test PilotFirefox测试飞行员，Firefox官方提供的各种新奇有趣的实验拓展。
Web DeveloperWeb开发调试工具，查看网页元素。
Sharp Color Picker取色器和颜色调整，适合网页开发。
Chrome Store Foxified让Firefox用上Chrome的扩展，但并不是所有都有使用。具体参考：这里。
Header Editor管理浏览器请求，包括修改请求头和响应头、重定向请求、取消请求。
Extension source viewer在不用安装Mozilla Addons 和Chrome Store 拓展的情况下查看拓展源码和打包下载。
