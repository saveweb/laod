关于Chrome 69 版本 一些改变以及设置

2018-09-08 16:26 • 工具软件 • 阅读 45710

不可否认，前几天Google更新Chrome 69版本感觉变化是最大的，很多童鞋都不能适应…

Chrome 69浏览器离线下载地址：

Stable 稳定版 32 位：69.0.3497.100  缓存服务器

Stable 稳定版 64 位：69.0.3497.100  缓存服务器

Chrome 69 更新的主要功能：

  1. 界面变化.
  2. 更聪明的搜索框-smarter Omnibox
  3. 默认启用画中画
  4. https 安全提示更新
  5. 默认禁用 flash
  6. 支持刘海屏和采用 AV1 视频解码

 

恢复 Chrome 浏览器原来的外观 UI

  1. 在浏览器地址栏输入 chrome://flags/#top-chrome-md 回车
  2. UI Layout for the browser’s top chrome 右边的选项修改为 Normal
  3. 重启浏览器

 

chrome 69 字体模糊、缩放模糊解决办法（69.0.3497.100版本已修复）

  1. 在浏览器地址栏输入 chrome://flags/#top-chrome-md 回车
  2. 将 UI Layout for the browser’s top chrome  右边的选项修改为 Normal
  3. 重启浏览器

 

Chrome 69 域名不显示 www 的解决办法（69.0.3497.100版本已默认显示）

  1. 在浏览器地址栏输入 chrome://flags/#omnibox-ui-hide-steady-state-url-scheme-and-subdomains 回车
  2. 将 Omnibox UI Hide Steady-State URL Scheme and Trivial Subdomains 右边的选项修改为Disabled
  3. 重启浏览器

 

Chrome 69 开启画中画功能

  1. 在浏览器地址栏输入 chrome://flags/ 回车
  2. 将 Experimental web platform features 右边的选项修改为 Enabled
  3. 将 Enable the use of SurfaceLayer objects for videos 右边的选项修改为 Enabled
  4. 将 enable picture-in-picture 右边的选项修改为 Enabled
  5. 重启浏览器

 

Chrome 69 网上应用店 & 扩展程序 无法发布评价

Chrome 69 网上应用店扩展的详情页改动非常大，几乎完全变成了应用商店风格，变得更美观了。不再是以往的原网页内嵌风格，而是调到新的页面。

扩展商店中文区已经无法发布评价，在右上角的小齿轮中设为英文可以发布。

 

Chrome 69浏览器界面返回老界面的方法

  1. 在浏览器地址栏输入 chrome://flags/ 回车
  2. 将 UI Layout for the browser’s top chrome 右边的选项修改为 Normal
  3. 重启浏览器

 

还有么？欢迎各位童鞋补充。
