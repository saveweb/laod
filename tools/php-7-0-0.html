PHP 7.0.0 正式发布，速度是 PHP 5.6 的两倍

2015-12-09 21:13

PHP 7 正式发布，PHP 7.0.0 包含最新版本的 Zend Engine，包括以下特性：
  • 提升性能：PHP 7 速度是 PHP 5.6 的两倍
  • 显著降低内存使用
  • 抽象语法树
  • 支持 64 位
  • 许多重大错误可以进行异常处理
  • 移除了旧的和不支持的 SAPIs 和扩展
  • 安全的随机数生成器
  • null 合并操作符(??)
  • 结合比较运算符 (<=>)
  • 标量类型声明
  • 匿名类
