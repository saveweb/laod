注：

1：系统需求：64位 Win7/Win10（部分Win10早期或部分版本不受支持）。

2：Adobe Creative Cloud 启用 PackageCode 管理新版程序（即HD），抛弃了 ProductCode，
故在 Adobe Creative Cloud 中不显示已安装。
本版为传统安装模式，不含Adobe Creative Cloud，也不接受Adobe Creative Cloud安装启动卸载。

3: 本版集成 Adobe Acrobat DC 特别版，
集成新版AMTLIB模拟破解(v0.9.2)。
集成最新版AAMv10(CS10)/ACCC SP。
移除试用提示。
移除自动更新，移除菜单更新。
打开、保存文件时不显示在线存储。
启动欢迎页不显示资讯。
最近使用列表中的最大文档数量默认为零。

4: Adobe Lightroom Classic CC 强制为订阅版，可使用去朦胧效果等完整订阅功能（许可证显示为试用，永久不支持订阅功能）。

5：ExtendScript Toolkit 和 Captivate 的简体中文/繁体中文语言包均由本人独立汉化完成。
原生模式，不影响其他语言的安装使用。禁止重打包发布。

6: CC 2018 支援各类新硬件、新特性，对硬件有较高的要求。极不推荐老旧设备上跑新版本，主观上的使用感受不如旧版本。

内存至少8GB、建议16GB及以上。
显存至少2GB、建议3GB及以上。

硬盘推荐SSD固态硬盘、NVME固态硬盘（推荐三星PM951、PM961、950PRO、960PRO及以上）。
软件缓存所在分区（默认为系统盘C:）保留足够的剩余空间（保持100G以上，尤其AE、PR根据实际需要更多）和高速读写支持。

内存、缓存空间不足，可制约您的高档CPU、出现低负载多核围观的场面。

请务必理解，高版本Adobe在处理复杂文件时需较高的配置以满足软件的内存和高速缓存请求、保证软件的响应速度。

建议1080P分辨率以上显示设备、推荐配备数位板/触摸屏。新版本对4K以上分辨率的新机型有更好的支持。

在出现迷之BUG时，请首先考虑在软件中关闭GPU硬件加速(包括 Camera Raw 的硬件加速)。

=====================================================================

安装流程：

百度云非常容易出现坏档，下载压缩包后注意校验。

压缩包包含5%的恢复记录(rr%5)，请尝试使用 WinRAR 修复损坏的分卷、改回原名后重新解压。

WinRAR 5.0以上版本解压；用虚拟光驱加载ISO，不要解压ISO。

解包而不使用镜像安装请务必注意不要将安装程序放置在过长的路径下导致的安装失败！

本版分为大师版和独立版，可根据个人需要安装。

大师版、独立版均为一键安装，一般无需任何操作。建议以管理员权限安装。

———————————————————————

注意：关于VC运行库问题
Microsoft Visual C++ 2005、2008、2010、2012、2013、2015 2017 Redistributable 32位、64位
针对很多非纯净系统环境下在安装过程中出现的VC运行库无法安装导致安装失败、安装程序错误，
建议先检查、卸载本地所有（2005、2008、2010、2012、2013、2015、2017）各种版本的VC运行库，
手动安装 .\Content\Runtime\ 目录下所有VC运行库后，再尝试安装程序。
启动程序提示缺少VC相关DLL问题，通常是VC运行库安装时未给予管理员权限、被误拦截或VC自身BUG(尤其VC 2015 R),
安装各类VC运行库合集包也解决此类问题(这些安装包不使用正规MSI安装方式，可以尝试)。

———————————————————————

有时在*安装前*需要清理残留的安装、激活信息——

一、使用官方清理工具先清理已安装程序
.\Content\Tools\Adobe Creative Cloud Cleaner Tool.exe

二、清理激活信息（删除文件、文件夹）
C:\ProgramData\Adobe\SLStore\
C:\Program Files (x86)\Common Files\Adobe\SLCache\
C:\ProgramData\regid.1986-12.com.adobe\

三、清理残留安装信息（删除文件、文件夹）
C:\Program Files (x86)\Common Files\Adobe\backup\
C:\Program Files (x86)\Common Files\Adobe\caps\
C:\Program Files (x86)\Common Files\Adobe\Installers\

四、检查本机AAM版本。Acrobat 许可模块安装失败，正版软件验证失败等问题请直接删除AAM文件目录。
（请注意查找并终止此目录下正在运行的进程，如AdobeIPCBroker.exe。原版 ACCC 桌面请先卸载。
已安装特别版 Adobe Creative Cloud 桌面，请同时终止Adobe Desktop Service.exe和其子进程）
C:\Program Files (x86)\Common Files\Adobe\OOBE\

———————————————————————

本版使用的是DLL PATCH + EMU 混合处理。默认全局屏蔽自动更新。

新版本安装、更新模式已变更，由ACC完全接管操作，想用ACC自动更新的请购买订阅。

手动更新：等待本特别版的更新，卸载旧版，安装新版即可。
