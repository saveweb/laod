如果来点音乐、再配上机械键盘的话，写代码是不是爽到爆呢？

（脑补播放动感的音乐再加上高大上的青轴键盘…最好准备好晕车药 :））

Windows 7安装

[cc lang=”python”]
cd .atom/packages
git clone https://github.com/JoelBesada/activate–power–mode.git
cd activate–power–mode
apm install
[/cc]
Mac安装

atom编辑器安装：
[cc lang=”php”]
sudo brew install Caskroom/cask/atom    #OS X 10.11.不支持通过命令行安装&可通过下载DMG文件安装
[/cc]
atom-install shell command
[cc lang=”php”]
cd /Users/$yourname/.atom/packages
git clone https://github.com/JoelBesada/activate–power–mode.git
cd activate–power–mode
apm install
apm ls #检测是否安装成功
[/cc]
