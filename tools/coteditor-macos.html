CotEditor 一款MacOS下好用的开源的文本编辑器

2018-12-10 16:53

老D在Windows平台下一直喜欢用Notepad++，但是到了MacOS平台一直没有找到好用、简单、轻量级纯文本的编辑器，今天看到一款叫CotEditor，果断下载回来试试~~~

CotEditor官网地址：https://coteditor.com

目前版本：3.6.8

CotEditor虽然是一款纯文本编辑器，但是各种语言的语法高亮、自动缩进、查找和替换正则表达式、命令行工具等功能还是很齐全的，CotEditor支持编辑C, C++, CSS, eRuby, HTML, Java, Javascript, LaTeX, Perl, PHP, Ruby, Shell 脚本等。

  • 支持语法高亮；
  • 强大的查找&替换功能；
  • 支持字符检测；
  • 支持窗口分割；
  • CJK 语言友好；
  • 支持用脚本语言编写 Macro
