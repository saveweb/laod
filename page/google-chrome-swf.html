Google chrome 浏览器播放.swf文件时变成了下载swf文件解决方法

2018-11-12 09:47

[email protected] ：谷歌打开带swf播放的网页时变成了下载swf文件，不知道怎么解决？

解决方案：

新建一个 html 文件，名称随意，比如 flash.html （可以新建记事本文本，然后改成html后缀即可）

然后输入如下的代码：

<html>
<head>
    <title>修复swf播放</title>
    <style type="text/css">
        embed
        {
            width:1540px;  // 设置长度
            height:722px;  // 设置高低
            }
    </style>
</head>
<body>
    <div>
    <embed src="文件路径"></embed>  // 在此输入文件路径或者运行swf文件的网页url
    </div>
</body>
</html>

然后用浏览器运行即可。

注：src 属性中的“文件路径”就是 .swf 文件的路径, 修改成自己需要的就行了，经过测试这个方法是好使的，而且比Potplayer 打开省资源。
